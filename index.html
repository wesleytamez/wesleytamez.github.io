Now can be interpreted!

  1. Download interpreter (.py file)

  2. Create folder
  3. Create program with .chess extension, following the grammar
  4. Run through CMD or your favorite IDE
  5. You're done! Make sure interpreter, program, and grammar are in the same folder.
